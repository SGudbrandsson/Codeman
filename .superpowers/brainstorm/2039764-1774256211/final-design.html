Implementation summary

Where: Two injected <div> elements added to the TranscriptView scroll container — not part of the JSONL block stream.

Trigger: SESSION_UPDATED SSE event (already fires on taskTree/subagent changes). Show task block when taskStats.running > 0, show agent block when any subagent is active.

Update: Re-render block contents in-place on each SESSION_UPDATED. No full transcript reload.

Remove: Task block removed when taskStats.running === 0. Agent block removed when all subagents are completed/idle.
